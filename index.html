Welcome to the Stratford-upon-Avon Memory Map!

Welcome to the Stratford-upon-Avon Memory Map, a project created as part of the 2021-2022 Cultural Heritage Informatics Fellowship. This map explores the various ways 19th century guidebooks described cultural heritage sites related to Shakespeare in his hometown. Click the swan icon for each location to view the visualization associated with that site. Be sure to also zoom out to check out Anne Hathaway's Cottage and Mary Arden's Farm!
